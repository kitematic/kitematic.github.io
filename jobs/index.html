Our Team

Kitematic (acquired by Docker in 2015) is a Mac app for managing Docker containers. It brings a visual interface to the Docker platform, making it even more powerful and easy to use. We're looking for an experienced software engineer to join the team. In this role, you'll leverage core pieces of the Docker platform to architect and build an intuitive product for developers.

Our team is highly autonomous. We love making developers' lives easier and every member is directly responsible for the product's success. Kitematic is open source and part of Docker, one of the fastest-growing open source communities in the world.

Position: Software Engineer

Responsibilities

  • Architect and launch new features end-to-end
  • Design & build high quality user interfaces that leverage key components of the Docker platform
  • Work closely with our Product and UI/UX teams to design solutions to problems faced by developers around the world

Requirements

  • Experience owning, implementing and launching core features of a product
  • Strong experience in Javascript
  • A passion for UI/UX, and for making developers' lives easier
  • Solid understanding computer science and programming fundamentals
  • Bonus: experience with Docker or building tools that use the Docker Remote API
  • Bonus: experience with Go

Potential Projects

  • A system for visually connecting and managing multi-container apps, powered by Docker Compose
  • A host management system for running and managing containers in many VMs or servers, powered by Docker Machine
  • Bringing Kitematic to Windows to more than double our developer reach
  • A button on DockerHub that immediately downloads and runs a container on your laptop in Kitematic
